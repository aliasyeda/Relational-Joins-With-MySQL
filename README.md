# Relational-Joins-With-MySQL


This project contains examples of different SQL JOINs using MySQL:

- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL JOIN (using UNION)

### Tables Used:
- employees
- departments

### Skills Practiced:
- Writing JOIN queries
- Understanding how data from multiple tables is combined
- SQL aliasing and matching keys

Created using MySQL Workbench 8.0 CE.
